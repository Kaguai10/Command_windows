# Modul 4 — File & Directory Operations

Modul ini membahas berbagai command Windows CMD yang digunakan untuk mengelola file dan directory seperti melihat isi folder, membuat folder, memindahkan file, menghapus file, hingga mencari lokasi file pada sistem Windows.

---

## Tujuan Pembelajaran

Setelah mempelajari modul ini, diharapkan dapat:

* Mengelola file dan folder melalui CMD
* Memahami operasi dasar directory Windows
* Melakukan navigasi file system
* Menggunakan command file management secara efisien

---

## Daftar Perintah

| Perintah    | Fungsi                       |
| ----------- | ---------------------------- |
| `dir`       | Menampilkan isi directory    |
| `dir /a`    | Menampilkan file hidden      |
| `tree`      | Menampilkan struktur folder  |
| `cd`        | Berpindah directory          |
| `mkdir`     | Membuat folder baru          |
| `rmdir`     | Menghapus folder             |
| `del`       | Menghapus file               |
| `copy`      | Menyalin file                |
| `move`      | Memindahkan file             |
| `rename`    | Mengubah nama file/folder    |
| `type`      | Menampilkan isi file         |
| `more`      | Membaca isi file per halaman |
| `attrib`    | Menampilkan atribut file     |
| `attrib +h` | Menyembunyikan file          |
| `where`     | Mencari lokasi file/program  |

---

## Catatan

* `dir /a` digunakan untuk melihat file hidden/system
* `tree` berguna untuk melihat struktur directory
* `attrib` digunakan untuk mengatur atribut file Windows
* `where` membantu mencari lokasi executable atau file
* Beberapa command membutuhkan hak Administrator
