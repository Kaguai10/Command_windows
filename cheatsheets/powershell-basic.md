# Modul 9 — PowerShell Basics

Modul ini membahas berbagai command dasar PowerShell yang digunakan untuk administrasi sistem, manajemen process, navigasi file, dan pengecekan informasi sistem pada Windows.

---

## Tujuan Pembelajaran

Setelah mempelajari modul ini, diharapkan dapat:

* Memahami command dasar PowerShell
* Melakukan administrasi sistem menggunakan PowerShell
* Mengelola file dan process
* Melihat informasi sistem Windows melalui PowerShell

---

## Daftar Perintah

| Perintah              | Fungsi                                |
| --------------------- | ------------------------------------- |
| `Get-Help`            | Menampilkan bantuan command           |
| `Get-Command`         | Menampilkan daftar command PowerShell |
| `Get-Process`         | Menampilkan process aktif             |
| `Stop-Process`        | Menghentikan process                  |
| `Get-Service`         | Menampilkan service Windows           |
| `Start-Service`       | Menjalankan service                   |
| `Stop-Service`        | Menghentikan service                  |
| `Get-ChildItem`       | Menampilkan isi folder                |
| `Set-Location`        | Berpindah directory                   |
| `Copy-Item`           | Menyalin file/folder                  |
| `Move-Item`           | Memindahkan file/folder               |
| `Remove-Item`         | Menghapus file/folder                 |
| `Get-Content`         | Membaca isi file                      |
| `Get-ComputerInfo`    | Menampilkan informasi sistem          |
| `Get-ExecutionPolicy` | Menampilkan policy PowerShell         |
| `Set-ExecutionPolicy` | Mengubah policy PowerShell            |

---

## Catatan

* `Get-Help` digunakan untuk melihat dokumentasi command PowerShell
* `Get-Process` mirip seperti `tasklist` pada CMD
* `Get-ChildItem` mirip seperti `dir` pada CMD
* `Set-ExecutionPolicy` sering digunakan saat menjalankan script PowerShell
* Beberapa command membutuhkan hak Administrator
