# Windows Terminal Notes

Repository ini berisi kumpulan command Windows CMD dan PowerShell yang disusun berdasarkan kategori agar lebih mudah dipelajari, dicari, dan digunakan kembali saat dibutuhkan.

Belajar command line Windows bukan hanya untuk administrator sistem atau praktisi IT profesional. Kemampuan memahami CMD dan PowerShell sangat berguna untuk troubleshooting, administrasi server, automation, digital forensics, penetration testing, hingga kebutuhan sehari-hari seperti mengelola file, jaringan, dan proses sistem dengan lebih cepat dan efisien.

Melalui repository ini, pembaca dapat memahami bagaimana Windows bekerja dari sisi terminal, mengenal command-command penting yang sering digunakan di dunia kerja, serta memiliki referensi praktis yang bisa langsung dipakai saat melakukan analisis, debugging, maupun operasional sistem. Semua materi disusun secara terstruktur agar mudah dipelajari baik oleh pemula maupun pengguna yang sudah berpengalaman.

Jika kamu ingin meningkatkan skill Windows, memahami proses sistem lebih dalam, atau membutuhkan catatan command yang rapi dan siap digunakan kapan saja, repository ini dibuat untuk membantu kebutuhan tersebut.
